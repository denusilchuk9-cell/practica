##Веб-застосунок для створення та проходження опитувань зі збереженням даних у localStorage.
## Функціонал

### 1. Створення опитування (Builder)
- Додавання питань двох типів: Multiple Choice та Text Field
- Редагування тексту питань
- Додавання/видалення варіантів відповідей
- Збереження опитування

## Інструкція з використання

1. **Створення опитування:**
   - Перейдіть на вкладку BUILDER
   - Введіть назву опитування
   - Натисніть "+ Multiple Choice" або "+ Text Field"
   - Заповніть текст питання та варіанти
   - Натисніть "PUBLISH"

2. **Проходження опитування:**
   - Перейдіть на вкладку TAKER
   - Виберіть опитування зі списку
   - Дайте відповіді на всі питання
   - Натисніть "SUBMIT"

3. **Перегляд результатів:**
   - Перейдіть на вкладку RESULTS
   - Перегляньте відповіді

## Технології

- HTML5
- CSS3
- JavaScript (Vanilla)
- LocalStorage для зберігання даних

## Збереження даних

Дані зберігаються в localStorage браузера:
- `surveys` - масив створених опитувань
- `lastResult` - останній результат проходження

## Ключові функції

- `addQuestion(type)` - додавання нового питання
- `publishSurvey()` - публікація опитування
- `selectSurvey(id)` - вибір опитування для проходження
- `submitSurvey(id)` - відправка відповідей
- `showResults()` - відображення результатів
## Висновок

Розроблено повнофункціональний застосунок для створення та проходження опитувань з інтуїтивним інтерфейсом та збереженням даних у браузері.
